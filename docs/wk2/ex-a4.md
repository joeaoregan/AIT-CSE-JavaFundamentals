## Java Fundamentals exercises
### Wk2 Ex A4

1. Write a program called CapitalLetter. Take in a letter (char) from the user. Using methods from the Character wrapper class, determine if the letter entered was a capital letter or not. If the letter is a capital letter output “Capital letter entered”; if a lowercase letter, output “Lowercase letter entered”; otherwise output an error (include the offending character in the error message).

> Enter a letter (a-zA-Z): *  
> Error in input: *  
> Enter a letter (a-zA-Z): a  
> Lowercase : a  
> The ASCII value of a is 97  
> Enter a letter (a-zA-Z): A  
> Uppercase : A  
> The ASCII value of A is 65  
> Enter a letter (a-zA-Z): h  
> Lowercase : h  
> The ASCII value of h is 104  
> Enter a letter (a-zA-Z): #  
> Error in input: #  
> Enter a letter (a-zA-Z): ;
